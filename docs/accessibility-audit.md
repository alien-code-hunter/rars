# Accessibility Audit

## Scope
- Global navigation, dashboards, repository, application workflows, admin portal.

## Implemented
- Skip-to-content link.
- Focus-visible ring styles for buttons/inputs (shadcn base).
- Clear text labels on all form controls.

## Recommended Follow-ups
- Verify color contrast on hero gradients and muted text.
- Ensure all icon-only buttons have `aria-label`.
- Validate keyboard navigation on dropdown menus and tabs.
- Run automated audit (Lighthouse) and fix remaining violations.
